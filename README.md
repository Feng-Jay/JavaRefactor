# JavaRefactor

JavaRefactor is a tool that can refactor java code while keeping original semantic.

## Supported Refactor Methods

- [x] RenameVariable
- [ ] BooleanExchange
- [ ] InsertLogStatement
- [x] LoopExchange
- [ ] ReorderStatement
- [ ] ReorderCondition
- [ ] SwitchToIf
- [ ] InsertUnusedStatement
